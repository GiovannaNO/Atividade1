1.git init: Serve para criar um novo repositório.
2.git config --global user.name: Serve para configurar seu nome de usuario, sua identificação.
3.git config --global user.email: Serve para configurar seu email.
4.git remote add origin: Serve para dar um destino ao qual o repositório é correspondente.
5.git status: Serve para verificar como os arquivos estão, o status de cada um.
6.git log: Serve para consultar histórico de alterações em ordem cronológica.
7.git add: Serve para adiciona todos os arquivos na área de preparação.
8.git commit : Serve para salvar as alterações no repositório local com a mensagem, commit.
9.git push: Serve para enviar as alterações feitas, os commits.
